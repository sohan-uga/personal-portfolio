Sohan Shankar's VR Portfolio

First Submission

Portfolio 1: DAWG Spawner
This demo is a more elaborate spawner in which hitting the keys 'D', 'A', 'W', 'G', or 'S' will spawn the corresponding letter. I created each of these letters in Unity by combining cubes into a parent object. This required knowledge of spawning objects, creating 3D objects in Unity, and keyboard input.
Portfolio 2: Pinwheel Material Changer
This demo drops cubes in a random direction. When the cubes come into contact with the pinwheel, they change to the color of the pinwheel before quickly dying. This required knowledge of triggers and materials.
Portfolio 3: Sticky Rockets
This demo has rockets fire in a random direction and has them eventually stick to a board. This required knowledge of colliders and how Rigidbodies work.
Portfolio 4: Raycast Gun Aimer
This demo has a randomly spawning board. A gun must use Raycast to find the distance to the board and shoot a projectile with a corresponding velocity. This required knowledge of the math behind projectiles and Raycasts.

Second Submission

Portfolio 5: Zombies
In this demo, I spawned GLTF zombie avatars that track the user and walk towards them, using a walking animation and a blended animation. I also implemented more movement functionality to MyMovement.cs. This required knowledge of avtars, animation, animation blending, and VR movement.
Portfolio 6: Fruit Samurai
In this demo, fruits are spawned, which a katana is then able to slice into two objects. I had to figure out the positioning and math of the sword to know where the cut is and identify the plane of the cut, and could then implement the base script that was used for cutting from 2D input. This required knowledge of plane math and decimating in Blender. The github for the mouse-based meshcutting is at https://github.com/KristinLague/Mesh-Cutting.
Portfolio 7: Boxing
In this demo, there is a fighter who can dodge your punches and then do a custom dance. Dodging the punches required using triggers and setting up an appropriate animation scheme. Creating the custom dance required the use of Avatar masks.
Portfolio 8: Hand Gestures
In this demo, I used transforms from the hand tracking skeleton to map out distances between key points on the right hand. I could then use these distances to determine the gesture being performed. This program recognizes a flat hand "hi" gesture, a thumbs up, and pointing with the index finger.
