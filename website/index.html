Sohan Shankar's VR Portfolio

Portfolio 1: DAWG Spawner
This demo is a more elaborate spawner in which hitting the keys 'D', 'A', 'W', 'G', or 'S' will spawn the corresponding letter. I created each of these letters in Unity by combining cubes into a parent object. This required knowledge of spawning objects, creating 3D objects in Unity, and keyboard input.
Portfolio 2: Pinwheel Material Changer
This demo drops cubes in a random direction. When the cubes come into contact with the pinwheel, they change to the color of the pinwheel before quickly dying. This required knowledge of triggers and materials.
Portfolio 3: Sticky Rockets
This demo has rockets fire in a random direction and has them eventually stick to a board. This required knowledge of colliders and how Rigidbodies work.
Portfolio 4: Raycast Gun Aimer
This demo has a randomly spawning board. A gun must use Raycast to find the distance to the board and shoot a projectile with a corresponding velocity. This required knowledge of the math behind projectiles and Raycasts.
